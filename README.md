# 🎓 Placement Predictor Pro

An AI-powered web application that predicts whether a student is likely to be placed based on their study hours and academic score. 

## ✨ Features
- **ML Powered**: Uses Logistic Regression to predict placement outcomes
- **Beautiful UI**: Glassmorphism + Neon theme with dark background
- **Real-time Prediction**: Get instant results with placement probability %
- **Unique Sidebar**: Royal purple gradient sidebar different from main background
- **User Friendly**: Simple 2-input interface for quick predictions
- **Responsive**: Works on both Mobile and Desktop

## 🛠️ Tech Stack
- **Frontend**: Streamlit, HTML/CSS
- **Backend**: Python
- **ML**: Scikit-learn, Pandas, NumPy
- **Model**: Logistic Regression with StandardScaler

## 📊 Model Details
| Parameter | Value |
| --- | --- |
| **Algorithm** | Logistic Regression |
| **Features** | Hours Studied, Academic Score |
| **Scaler** | StandardScaler for feature normalization |
